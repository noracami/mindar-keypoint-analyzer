# MindAR 特徵點分散度分析工具 — 設計文件

## 架構

單頁應用（SPA），一個 `index.html` + 一個 `main.js` + 一個 `style.css`，用 Vite 打包。

## UI 佈局（簡潔功能導向）

頁面由上往下，按步驟線性排列：

1. **上傳區** — 拖放區域 + 檔案選擇按鈕，下方顯示已上傳圖片的縮圖與檔名
2. **編譯區** — 「開始編譯」按鈕 + 進度條，完成後顯示「下載 .mind 檔」連結
3. **分析報告區** — 編譯完成後自動顯示：
   - 摘要表格（特徵點數、獨有數、共用數、獨有率 %）
   - N×N 重疊矩陣表格
   - 結論判定（綠/黃/紅色標示）
   - 可展開的 JSON 原始資料區（附複製/下載按鈕）

## 特徵點取得方式

從 `compiler.data` 中取得每張圖的 matching data。MindAR Compiler 編譯後，`compiler.data` 會包含多層級的特徵點資訊，取最高解析度層的特徵點座標與 FREAK descriptor。

## 分析邏輯

- 正規化座標為 0~1
- 雙重條件判定重疊：座標歐氏距離 < 1% **且** FREAK descriptor Hamming distance < 40%（266/666 bits）
- 計算 N×N 重疊矩陣與獨有率
- FREAK descriptor 為 666 bits，以 21 個 32-bit 整數打包，具旋轉不變性

## 判定標準

- 獨有率 > 50%：辨識度良好（綠色）
- 獨有率 30%~50%：辨識度中等（黃色）
- 獨有率 < 30%：辨識度差（紅色）

## 視覺化

`src/visualize.js` 提供特徵點視覺化：

- 個別圖片的特徵點分佈（綠色＝獨有、紅色＝共用）
- 配對重疊比較（紅點＝重疊特徵點、藍框＝密集重疊區域）

## 技術棧

- Vite + Vanilla JS
- `mind-ar` npm 套件
- 純手寫 CSS，白底、系統字體、最小裝飾
- 部署至 Cloudflare Pages
